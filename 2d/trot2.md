# trot2
_SE(2) rotation matrix_


```T = TROT2(THETA)``` is a homogeneous transformation (3&times;3) representing a rotation
of `THETA` radians.


```T = TROT2(THETA, 'deg')``` as above but `THETA` is in degrees.
### Notes
* Translational component is zero.
